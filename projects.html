Engineering Projects

This section presents a comprehensive overview of my engineering projects, spanning embedded hardware design, intelligent sensing systems, computer vision, applied machine learning, and autonomous platforms. Each project reflects hands-on development, system-level integration, and engineering decision-making under real technical constraints.

Image-Driven Foam Stability Analysis

Microscopy Hackathon 2025 — AISCIA Informatics

Developed an image-driven analytical pipeline to study bubble dynamics and foam stability using time-resolved microscopy images. The objective was to extract physically meaningful stability indicators directly from visual data without intrusive measurements.

Designed segmentation and tracking workflows to quantify bubble size distributions, coarsening rates, spatial packing, and entropy-based disorder metrics. These features were integrated into an explainable machine learning framework to correlate early-stage dynamics with long-term foam stability.

Technologies: Python, image segmentation, feature engineering, explainable ML, scientific data analysis

Custom ESP32 Board Design

Hardware Design Project

Designed a custom ESP32-based development board to gain full-cycle hardware experience from schematic design through PCB layout and validation.

Integrated ESP32-WROOM module, CP2102 USB–UART bridge, AMS1117 voltage regulation, ESD protection, and USB differential pair routing on a four-layer PCB. Generated complete manufacturing outputs including BOM and pick-and-place files.

Technologies: ESP32, EasyEDA, 4-layer PCB design, USB, hardware debugging

STM32-Based Embedded Hardware Platform

PCB & Embedded Systems Project

Designed a custom embedded platform around the STM32F103 microcontroller with focus on reliable power delivery, clocking, and peripheral integration.

Implemented decoupling networks, external oscillator circuitry, USB connectivity, and UART/I²C interfaces following datasheet and signal integrity best practices. Generated fabrication-ready Gerbers and BOMs.

Technologies: STM32F103, KiCad v9, embedded hardware design

SP02Sense — Arduino-Based Pulse Oximeter

Embedded & Biomedical Sensing Project

Designed a compact pulse oximeter to measure blood oxygen saturation and heart rate using the MAX30102 optical sensor.

Developed a four-layer PCB integrating the MAX30102 sensor, SSD1306 OLED display, debounced buttons, and LED-based alert system. Implemented real-time firmware for data acquisition, signal processing, and user interaction.

Technologies: Arduino, MAX30102, OLED display, real-time embedded firmware

Fuel Blend Property Prediction

Shell.ai Hackathon 2025

Built predictive machine learning models to estimate ten fuel properties for conventional and sustainable aviation fuel (SAF) blends.

Engineered polynomial and interaction features, implemented stacking ensembles combining LightGBM, XGBoost, and Ridge regression, and optimized performance through cross-validated hyperparameter tuning.

Technologies: Python, LightGBM, XGBoost, feature engineering, regression modeling

Customizable Hand Gesture Controller for Gamers

Computer Vision & Human–Computer Interaction Project

Developed a real-time hand gesture recognition system for gaming applications using camera-based input.

Implemented gesture detection and tracking using MediaPipe and OpenCV, and trained classification models with TensorFlow and Keras. Designed a modular architecture allowing users to define custom gestures mapped to in-game actions.

Technologies: OpenCV, MediaPipe, TensorFlow, Keras, real-time vision

Autonomous Vehicle-Tracking Drone System

SCORPE Tech — Industry Project

Contributed to the development of an autonomous drone system capable of real-time vehicle tracking and pursuit using onboard computer vision.

Integrated a YOLOv8-based detection pipeline with the drone’s onboard compute and flight-control stack, achieving low-latency inference and high autonomy. Implemented camera calibration, route generation, and LiDAR-based obstacle avoidance, resulting in zero collision incidents during testing.

Technologies: YOLOv8, embedded AI, drones, LiDAR, autonomous navigation

Smart IoT Traffic System — ALPR Gate

Qatar University Capstone Project

Built a smart traffic gate system using automatic license plate recognition to reduce congestion and improve access control efficiency.

Implemented an ALPR pipeline using Raspberry Pi and Rekor Scout, achieving over 95% recognition accuracy within 10 meters. Designed a physical gate prototype driven by a NEMA 23 stepper motor and DM542T driver, reducing gate congestion by approximately 60%.

Technologies: Raspberry Pi, ALPR, stepper motors, IoT
